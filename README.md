Delicious Meals Restaurant Project Instructions
1. Project Title
Delicious Meals Restaurant - Web Project

2. Project Description
This project is a web-based system designed to manage orders for a restaurant named "Delicious Meals Restaurant". It provides a user-friendly interface for customers to browse various menus (food, juices, desserts), view special offers, manage their orders, and contact the restaurant. The system is built using fundamental web technologies (HTML, CSS, JavaScript) and focuses on client-side functionality, including local storage for order management.

3. Main Features
The key features of the Delicious Meals Restaurant project include:

•	User Login: A simple login page (index.html) to access the main application, redirecting to the home page upon successful (simulated) login.
•	Comprehensive Navigation: A clear navigation bar allowing users to easily access different sections of the restaurant's website.
•	Diverse Menus: Dedicated pages for the main food menu, juice menu, and desserts menu, each displaying various items with images and prices.
•	Special Offers: A section to showcase current promotions and special deals.
•	Order Management: Users can add items from any menu to their order list. The order details are stored locally using localStorage.
•	View and Clear Orders: A dedicated "My Orders" page where users can view their selected items and the total price, with an option to clear their current order.
•	Contact Form: A "Contact Us" page with a form for customer inquiries and contact information.
•	Responsive Design: The website is designed to be accessible and functional across various devices and screen sizes.

5. Technology Stack & Tools Used
This project is developed using standard web technologies:

•	HTML5: Used for structuring the content and defining the layout of all web pages.
•	CSS3: Applied for styling the website, ensuring an appealing visual presentation and responsive behavior across different devices. The main stylesheet is css/style.css.
•	JavaScript (ES6+): Implements client-side interactivity, including the login functionality, adding items to the cart, and managing orders using the browser's localStorage. The main script file is js/script.js.
•	Development Tools: A standard web browser (e.g., Chrome, Firefox) for viewing and testing, and a code editor (e.g., Visual Studio Code) for development.

How to Run the Project
1	Open index.html: Navigate to the extracted Delicious Meals Restaurant folder and open the index.html file in your web browser. This will take you to the login page.
2	Login: Enter any username and password (the login is simulated and does not require specific credentials) and click "دخول" (Login) to proceed to the home page.
3	Explore: Use the navigation bar to explore the different menus, offers, and manage your orders.
